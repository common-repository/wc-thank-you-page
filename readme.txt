WC Thank You Page redirect internal custom thank you page or external any url after completing orders  on your WooCommerce shop and make your users feel pampered.

== Description ==
Install the plugin to set up a custom Thank You page to show to your customers after they complete an order. You can use any WordPress page or external URL of your choice and customize it as you prefer, so to give your own personal touch of style to the website and to fill a small gap in the purchasing process: saying thanks to them!

= Features of WC Thank You Page: =

* Set a custom Thank you page for all products
* Set a Custom Thank You page for per Product.
* You can design any page using page builder like Elementor, Divi etc.

= Supported Page Builder: =

* Elementor
* Divi Builder
* VisualComposer
* Oxizen Builder
* Beaver Builder
* Etc
Please use this page builder's text editor and use shortcodes.

= Shortcodes: =

* You can use this shortode to design thank you page using any page builder ( Elementor, Divi, VisualComposer etc ) code eidtor or WordPress default code editor

[wpte_order_number]
[wpte_first_name]
[wpte_order_status]
[wpte_currency]
[wpte_total_price]
[wpte_created_date]
[wpte_shipping_method]


== Installation ==

Important: First of all, you have to download and activate WooCommerce plugin, because without it, WC Thank You Page cannot work.

* Unzip the downloaded zip file.
* Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
* Activate `WC Thank You Page` from Plugins page


= Configuration =

* Configure the plugin at Dashboard > WC Thank You.
